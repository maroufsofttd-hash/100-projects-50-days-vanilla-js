# 🔥 100 Projets JavaScript Vanilla — Marouf

## 🎯 L'objectif

Construire 100 petits projets en JavaScript vanilla, du plus simple au plus complexe, pour maîtriser les fondamentaux de bout en bout. Pas de React, pas de Vue, pas de bibliothèques magiques — juste HTML, CSS et JS pur.
